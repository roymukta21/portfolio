# Mukta Roy - Web Developer Portfolio

A modern, animated portfolio website showcasing the work and skills of Mukta Roy, a passionate full-stack web developer.

## ✨ Features

### 🎨 **Unique MR Logo**
- Custom-designed animated logo with "MR" initials
- Gradient background with floating particles
- Rotating decorative elements
- Shimmer text effects
- Hover animations with 360° rotation

### 🚀 **Modern Animations**
- **Framer Motion** for smooth React animations
- **GSAP with ScrollTrigger** for scroll-based animations
- **Lenis** for buttery smooth scrolling experience
- Staggered animations for better visual flow
- Interactive hover effects throughout

### 🎯 **Design Highlights**
- **Deep Yellow Color Scheme** - Warm, professional gradient from yellow-400 to orange-500
- **Glass Morphism Navigation** - Backdrop blur effects when scrolled
- **Animated Skill Bars** - Progress bars with shimmer effects
- **Floating Elements** - Subtle background animations
- **Responsive Design** - Mobile-first approach

### 📱 **Sections**
- **Hero Section** - Animated introduction with gradient text
- **About Me** - Personal story with animated contact info
- **Services** - Frontend Development, Backend Development, Full-Stack Solutions
- **Skills** - Animated progress bars for Frontend, Backend, and Development Tools
- **Portfolio** - Featured web development projects with hover effects
- **Contact** - Contact form and information

## 🛠 **Technologies Used**

- **React 18** - Modern React with hooks
- **Framer Motion** - Production-ready motion library
- **GSAP** - Professional animation library
- **Lenis** - Smooth scrolling library
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - High-quality, accessible UI components
- **Material Symbols** - Modern icon system
- **Class Variance Authority** - Component variant management
- **Tailwind Merge** - Utility class merging

## 🚀 **Getting Started**

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm start
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## 🎨 **Customization**

### Logo Sizes
The Logo component supports different sizes:
- `sm` - Small (32px)
- `md` - Medium (40px) - Default
- `lg` - Large (48px)
- `xl` - Extra Large (64px)

### Color Scheme
Primary colors are defined in CSS variables:
- `--color-primary`: #f59e0b (Yellow)
- `--color-secondary`: #ea580c (Orange)
